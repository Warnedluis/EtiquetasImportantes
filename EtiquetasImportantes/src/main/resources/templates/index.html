Definicion de HTML

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el estándar fundamental para crear y estructurar el contenido de las páginas web. Utiliza una serie de etiquetas (tags) para organizar elementos como párrafos, títulos, imágenes, enlaces y videos, actuando como el esqueleto que estructura el contenido para los navegadores web

Etiquetas mas importantes en HTML

HTML como tal se sirve de etiquetas, las cuales definen las partes de una pagina web. Dichas etiquetas se valen de diferentes usos, siendo estas las siguientes divididas en una pagina web cada una

  1. Texto
  2. Formularios
  3. Imagenes
  4. Listas
  5. Multimedia
  6. Semanticas
  7. Tables
