Etiquetas de listas en HTML

Uno de los aspectos mas importantes en HTML, es el poder formar listas, ya que esto nos permite organizar de mejor manera elementos

En el caso de las listas, hay dos tipos importantes de ellas, las listas ordenadas y desordenadas

Listas desordenadas en HTML

Las listas desordenadas en HTML son aquellas en las que se listan elementos mediante solo puntos o algun otro tipo de figura, excepto numeros o algo que lleven un orden

Un ejemplo de dichas listas seria la siguiente

\
  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4

Listas ordenadas en HTML

Las listas ordenadas en HTML son aquellas que a diferencia de las desordenadas, si cuentan con elementos ordenados por numeros

Un ejemplo de dichas listas es el siguiente

  1. Elemento 1
    2. Elemento 2
  2. Elemento 3
  3. Elemento 4
  4. Elemento 5
