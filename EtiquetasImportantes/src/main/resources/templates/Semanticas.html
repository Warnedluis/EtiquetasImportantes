Semantica en HTML

La semantica en HTML permite la estructuracion del contenido describiendo su significado tanto para navegador como para los desarrolladores.

Diagrama de la semantica de una pagina web

Header

La etiqueta header representa un contenedor para contenido introductorio o enlaces de navegacion

Footer

La footer se situa al final de una seccion o pagina, conteniendo informacion sobre el autor o contacto

Nav

La etiqueta Nav define un bloque de enlaces de navegacion principal, como un menu o indice

Main

La etiqueta main contiene el contenido central y unico de la pagina web

Section

La etiqueta Section permite agrupar el contenido relacionado semanticamente, generalmente con su propio encabezado, para poder dividir la pagina en secciones tangibles

Article

La etiquetqa article define contenido autonomo, independiente y reutilizable, como una entrada de blog, noticia o comentario
